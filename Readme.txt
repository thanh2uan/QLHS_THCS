Bài nộp bao gồm: 
	1. Thư mục "BaoCao": chứa file báo cáo.	
	2. Thư mục "FileThucThi": chứa file thực thi chương trình.
	3. Thư mục "SourceCode": chứa project cài đặt chương trình.
	4. Thư mục "ThuVien_Plugin": chưa file cài đặt plug-in DEV EXPRESS 14.2.3.
	5. Thư mục "DuLieuMau": các file execel dữ liệu mẫu, thư mục file script SQL thiết kế cơ sở dữ liệu.
Lưu ý:
	1. File source code sử dụng Visual studio 2013.
	2. Trước khi sử dụng cần cài đặt DEV EXPRESS.
	3. Chương trình sử dụng .net Framework 4.5.
	4. Sử dụng các file dữ liệu mẫu đúng thông tin. Ví dụ: trong file xếp lớp có thông tin học kỳ - năm học,
	   lớp thì trong cơ sở dữ liệu cần có các thông tin này trước.
